ESOE 5089 Design and Application of Virtual Instruments

LabVIEW is a graphical programming environment commonly used for data acquisition, instrument control, and industrial automation. Different from its general use, I recreated a mini-game coding in LabVIEW. The original game is called Pikachu volleyball, which I enjoyed playing when I was a child.

All of the movable objects in the game display were created from boolean buttons. The most challenging part in this project was to design and designate the bouncing mechanism of the ball, including its speed and direction.
